# Task 1 - Data Immersion & Wrangling

## Objective
The goal of this project is to perform data immersion, data quality assessment, data cleaning, transformation, and feature engineering using Python and Pandas.

---

## Dataset Used
Sales Dataset (sales_data_sample.csv)

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

---

## Steps Performed

### 1. Data Understanding
- Loaded dataset
- Explored rows and columns
- Analyzed dataset structure

### 2. Data Quality Assessment
- Checked missing values
- Checked duplicate rows
- Analyzed unique values

### 3. Data Cleaning
- Filled missing values
- Removed duplicates
- Converted date formats

### 4. Feature Engineering
- Extracted order month
- Extracted order year

### 5. Data Visualization
- Sales distribution
- Country-wise order analysis
- Deal size distribution

---

Cleaned dataset saved as:
cleaned_sales_data.csv
